# My Virtual Bookshelf

I wanted a Spotify-wrapped version for the books I read this year and I also had a bit of nostalgia for my tumblr blog that I maintained back in 2014. What resulted is (lea.earth)[https://lea.earth/] 

## Features

- Books organized by year with filtering
- 3D flip cards that reveal title and author on hover
- Year analytics and collections
- Favorites section
- Fully responsive design

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Project Structure

```
src/
├── components/     # React components
├── data/          # books.json with book data
└── utils/         # Utility functions
```

## Technologies

- React 18
- Vite
- Tailwind CSS
