# My Reading Journey

A React website showcasing books I've read from 2020-2025, featuring 3D flip card animations and a purple aesthetic.

## Features

- Books organized by year with filtering
- 3D flip cards that reveal title and author on hover
- Year analytics and collections
- Favorites section
- Fully responsive design

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Project Structure

```
src/
├── components/     # React components
├── data/          # books.json with book data
└── utils/         # Utility functions
```

## Technologies

- React 18
- Vite
- Tailwind CSS
